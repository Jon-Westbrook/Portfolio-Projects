Typography

The CardX typography system uses the Roboto font family in a range of display classes to control the size of any font. While default sizes are applied to hierachical elements like h1, h2, h3, for the sake of consistency it is preferred to style all text with a display size class (eg. display1) and an optional modifier (eg. small).

Display Headings

Using a display modifier is the preferable way to change the size of any text element.

Code Example

class="display1"

Display 1 - 48px / 52px - Light

class="display2"

Display 2 - 36px / 41px- Light

class="display3"

Display 3 - 32px / 37px - Regular

class="display4"

Display 4 - 28px / 33px - Regular

class="display5"

Display 5 - 24px / 29px - Regular

class="display6"

Display 6 - 14px / 18px - Medium

class="display7"

Display 7 - 12px / 16px - Medium

HTML Headings

HTML Headings have default styles that correlate directly to the display sizes. H1 elements are equivalent to Display 3 by default. Adding any display class to an HTML element will override the display

Code Example

<h1></h1>

 h1. Heading Roboto Regular

<h2></h2>

 h2. Heading Roboto Regular

<h3></h3>

 h3. Heading Roboto Regular

<h4></h4>

 h4. Heading Roboto Regular

<h5></h5>

 h5. Heading Roboto Medium Uppercase

Style Modifiers

The style modifiers can be applied to any text element, like an HTML Heading <h1></h1>, <span></span> or <p></p>.

Code Example

<p></p>

Primary Text

<p class='text-secondary'></p>

Secondary Text

<p class='small'></p>

Small Text

<a href='#'></a>

 Link Text

<label></label>

<p class='text-success'></p>

Success Text

<p class='text-danger'></p>

Danger Text

<p class='text-warning'></p>

Warning Text

<p class='text-info'></p>

Info Text

<p class='text-light'></p>

Light Text

<p class='text-dark'></p>

Dark Text

<p class='text-muted'></p>

Muted Text

<p class='text-white'></p>

White Text

Code

An inline code element can be applied to any programmatic text, like <h1></h1>, <span></span> or <p></p>.

A block of code can be styled using a the code-block class:

<p>Sample text here...</p>
<p>And another line of sample text here...</p>
