Color Palette

The CardX color palette has been choosen around their two primary colors. The primary, secondary and tertiary palettes help create a color theme that is harmonious and accessible, and helps distinguish UI elements and backgrounds from one another.

Code Usage

Colors are applied via color classes corresponding to typography and backgrounds.

Typography - use class="[color-name]". Typography-specific utilities that affect color are available and documented under typography.

Background - use class="bg-[color-name]".
